# שעות עבודה - השכר שלי

אפליקציית מעקב שעות עבודה ושכר, חלופה נקייה וללא פרסומות לאפליקציות הקיימות.

## ✨ תכונות

- ⏱️ **כניסה/יציאה למשמרת** - כפתור פשוט וגדול למעקב זמן בזמן אמת
- 📊 **יומן משמרות** - רשימה מלאה של כל המשמרות עם אפשרות עריכה ומחיקה
- 💰 **חישוב שכר אוטומטי** - לפי חוקי עבודה ישראליים
  - שעות נוספות (125%, 150%)
  - תוספת שבת/חג (150%)
  - דמי נסיעות (ליום או חודשי)
- 📈 **דוח חודשי מפורט** - פירוט מלא של ההכנסות
- ⚙️ **הגדרות מתקדמות** - התאמה אישית מלאה
- 🌙 **מצב כהה** (בקרוב)

## 🚀 התקנה והרצה

```bash
# התקנת dependencies
npm install

# הרצה במצב פיתוח
npm run dev

# בניית האפליקציה לפרודקשן
npm run build

# הרצת בנייה מקומית
npm run preview
```

## 🛠️ טכנולוגיות

- **React 18** - ספריית UI
- **TypeScript** - type safety
- **Vite** - build tool מהיר
- **Tailwind CSS** - עיצוב
- **React Router** - ניווט
- **Lucide React** - אייקונים
- **localStorage** - שמירת נתונים (ללא צורך בשרת)

## 📱 תמיכה

האפליקציה מותאמת למובייל (mobile-first) ועובדת היטב גם על דסקטופ.

## 🔒 פרטיות

כל הנתונים נשמרים **רק במכשיר שלך** (localStorage). אין שרת, אין שליחת מידע, פרטיות מלאה.
